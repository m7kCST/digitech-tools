☁️ OneDrive folders 📛 File naming 🗂️ Folder structure ⚡ Do it once, sorted forever
Why does this matter? When your files are named correctly and saved in the right folder, you can find them instantly, your teacher can see your work, and you never lose anything. This guide shows you how to set up your OneDrive once so it works every time for the rest of the year.

The Naming Rule — Every File, Every Time 📛

Every filename follows this exact format:
2026-03-04_DigiTech_Classwork_My-Document.docx
Today's date (YYYY-MM-DD)
Subject name
Type of work
Short title (no spaces — use hyphens)
File extension
2026-03-04_Science_Notes_Cells-and-Organisms.docx
2026-03-04_English_Assessment_Short-Story-Draft.docx
2026-03-04_HASS_Presentation_Ancient-Egypt.pptx
2026-03-04_DigiTech_Classwork_Budget-Spreadsheet.xlsx
💡 Use the File Name Wizard

Go to the class site and use the File Name Wizard — it generates the correct filename for you automatically. Just fill in the subject, type and title, then copy and paste.

How to Create the Folders 🛠️

📁
Step by Step — Do This Once
Takes about 5 minutes. Never have to do it again.
1
Open OneDrive in File Explorer
Click the cloud icon ☁️ in your taskbar — or open File Explorer and find OneDrive on the left sidebar.
2
Create a folder called "Year 7"
Right-click in an empty space → New → Folder → type Year 7 → press Enter
3
Open Year 7 — create a folder for each subject
Double-click Year 7. Right-click → New → Folder for each subject: DigiTech, English, Maths, Science, HASS, Health-PE, Food-Tech, Art, Music, Religion.
4
Inside each subject — create 3 folders
Open each subject folder and create: Classwork, Assessments, Resources. Repeat for every subject.
OneDrive → Year 7 → Maths → 📁 Classwork
                                → 📁 Assessments
                                → 📁 Resources
5
Pin "Year 7" to Quick Access
Right-click the Year 7 folder → Pin to Quick Access. Now it appears in the left sidebar every time you open File Explorer — one click to get there.

Do This / Never Do This 🚦

❌ Never do this
  • Leave files on your Desktop
  • Use spaces in filenames
  • Use emojis in filenames
  • Name files "Untitled" or "Document1"
  • Save everything to Downloads
  • Skip renaming files from your teacher
✅ Always do this
  • Name the file before you open it
  • Use hyphens - instead of spaces
  • Keep titles short (2-4 words)
  • Save in the right subject folder
  • Use the File Name Wizard if unsure
  • Rename files your teacher gives you

The Rules That Actually Matter ⭐

⭐ Golden Rules

1

Always use the File Name Wizard before opening any app. Name first, save correctly, then start working. Never rename files after the fact.

2

No spaces in filenames. Use hyphens (-) between words in the title section. Spaces break file links and cause problems in some apps.

3

Save to the correct subject folder every time. A Maths file in the DigiTech folder is just as lost as a file on the Desktop.

4

The date goes first — in the format YYYY-MM-DD. This keeps files sorted in date order automatically when you browse your folder.

5

If your teacher gives you a file, rename it using the naming convention before you start editing it. Don't leave it as whatever they called it.
