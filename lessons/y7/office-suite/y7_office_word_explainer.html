What Even Is Word? 🤔

📄

A Word Processor

Word lets you create, edit and format text documents. Essays, letters, reports, resumes — if it's a document with words, Word is the tool for it.

🌍

Used Everywhere

Schools, offices, hospitals, law firms — everyone uses Word. Knowing how to use it properly is one of the most useful skills you'll pick up this year.

🎨

Formatting = Clarity

Good formatting makes documents easier to read and more professional. Headings, spacing, fonts — these aren't decorations, they're communication tools.

📁

Files = Documents

A Word file is called a document. Save it as .docx and store it in your OneDrive folder using the correct filename from the File Name Wizard.

Key Terms — Know These 📍

Ribbon
Home / Insert / Layout

The toolbar at the top with tabs. Each tab (Home, Insert, etc.) shows different groups of tools relevant to that task.

Font
Calibri, Arial, Times

The style of the letters. Calibri is Word's default — clean and readable. Change it in the Home tab. Stick to one font for the whole document.

Font Size
12pt body / 16pt heading

How big the text is, measured in points (pt). Body text is usually 12pt. Headings are bigger — 14–18pt depending on the level.

Paragraph Style
Normal / Heading 1 / Heading 2

Pre-set formatting applied to a whole paragraph. Styles control font, size, colour and spacing all at once — much faster than formatting manually.

Line Spacing
1.0 / 1.5 / 2.0

The gap between lines of text. Single spacing (1.0) is compact. Double spacing (2.0) is common for assignments. Set it in Layout or Home.

Margins
Layout tab > Margins

The blank space around the edges of the page. Default is 2.54cm (1 inch) on all sides. Don't shrink them to squeeze more text in.

Navigation Pane
View > Navigation Pane

A panel showing all your headings as a clickable outline. Click any heading to jump straight to that section — essential for long documents.

Word Count
Bottom status bar

Shows how many words are in your document. Click it for a detailed breakdown. You can also select text first to count just that section.

Text Formatting ✏️

Ctrl + B
Bold
This text is bold

Use for headings, key terms, or anything that needs to stand out. Don't bold random words for decoration.

Ctrl + I
Italic
This text is italic

Use for titles of books/films, technical terms, or subtle emphasis. Avoid whole paragraphs in italics.

Ctrl + U
Underline
This text is underlined

Rarely needed in modern documents — links are underlined automatically. Use sparingly if at all.

Ctrl + E
Centre Align
Centred text

Good for titles and headings. Don't centre body text — left-align is easier to read.

Ctrl + L
Left Align
Left-aligned text

The default for body text. Left-aligned text is easier for the eye to track line by line.

Ctrl + Z
Undo
↩ Undo last action

Undoes your last action. Press multiple times to go back further. Your best friend when something goes wrong.

Paragraph Styles & Headings 🏷️

Why Use Styles Instead of Manual Formatting?

Instead of manually making text big and bold for every heading, apply a Style — a saved set of formatting. Change the style once and every heading updates automatically. It also enables the Navigation Pane and Table of Contents features.
Title Document Title 28pt bold · once per doc

Heading 2 Sub-section 14pt · sub-sections
Heading 3 Sub-sub-section 12pt bold · minor sections

💡 How to apply: Click anywhere in a paragraph → go to Home tab → click the style name in the Styles gallery. Done. No manual font sizing needed.

Tables 📊

When and How to Use Tables

Tables organise data into rows and columns. Use them for comparing things, showing schedules, or presenting data clearly. Don't use tables just to make text sit side-by-side — that's what columns are for.

How to insert a table
1
Click where you want the table in your document.
2
Go to Insert tab → click Table.
3
Drag over the grid to choose rows and columns, or click Insert Table to type the numbers.
4
Click inside any cell and start typing. Press Tab to move to the next cell.
5
Use Table Design and Layout tabs to style and resize.
💡 Tab key trick: Pressing Tab in the last cell of the last row automatically adds a new row. No right-clicking needed.

Page Layout 📐

Margins
Layout > Margins

Default is Normal (2.54cm all around). Use Narrow only if you genuinely need more space. Don't use tiny margins to cheat word counts.

Orientation
Layout > Orientation

Portrait (tall) is the default. Landscape (wide) is useful for wide tables or diagrams. Most documents stay Portrait.

Line Spacing
Home > Line Spacing

Single (1.0) for normal documents. 1.5 or Double (2.0) for assignments that need space for marking. Set it for the whole document before you start typing.

Page Breaks
Ctrl + Enter

Forces the next content onto a new page. Use this instead of hammering Enter lots of times — pressing Enter to create space will break your formatting.

Headers & Footers
Insert > Header / Footer

Text that appears on every page — your name, date, page numbers. Double-click at the top or bottom of a page to edit them.

Page Numbers
Insert > Page Number

Auto-numbers every page. Goes in the footer usually. Essential for any document longer than 2 pages.

Step 3 — Ctrl+S always
Every 5 minutes

Save constantly. If the laptop dies or Word crashes and you haven't saved, that work is gone. No exceptions.

Check the title bar
Should show OneDrive

The filename at the top should include your correct name. If it says Document1 or shows a Desktop path, you haven't saved correctly yet.

Printing Basics 🖨️

Print Preview
Ctrl + P

Always preview before printing. Ctrl+P opens the print dialog with a preview. Check all pages look right before sending to the printer.

Printer selection
Choose the right printer

Make sure the correct printer is selected at the top. Sending to the wrong printer means walking around trying to find your document.

Page range
All / Current / Custom

Print all pages, just the current page, or a range (e.g. 2-4). Don't print 20 pages when you only need page 3.

Print layout check
Before you print

Check margins, check no text is cut off, check headings are on the right pages. Fix any issues before printing — paper is not free.

The Rules That Actually Matter ⭐

⭐ Golden Rules

1

Use Styles for headings — never manually bold and resize text. Styles give you the Navigation Pane, consistent formatting, and the ability to update everything at once.

2

Never press Enter multiple times to create space. Use paragraph spacing (Home > Line Spacing > Add Space After Paragraph) or a page break (Ctrl+Enter).

3

Stick to one font for the whole document. Calibri or Arial for body text is standard. Don't mix Comic Sans and Times New Roman.

4

Use the Navigation Pane for any document longer than one page. View tab > Navigation Pane. Click headings to jump around instantly.

5

Name your file using the File Name Wizard before you open Word. Save to OneDrive. Ctrl+S every few minutes. No excuses.

6

Always preview before printing (Ctrl+P). Check every page looks right. Printing a broken document and reprinting it is a waste of everyone's time and paper.

When Things Go Wrong 🚨

🚨 Common Mistakes

Spacing looks wrong

Fix: You've probably been pressing Enter to add space. Select the paragraph, go to Home > Line Spacing > Remove Space After Paragraph. Then set proper spacing in the style settings.

Navigation Pane is empty

Fix: You haven't used Heading styles. The Navigation Pane only picks up text formatted with Heading 1, 2 or 3 styles — not manually bolded text. Apply the styles from the Home tab.

Text is in a weird font

Fix: Select all (Ctrl+A), then set the font in the Home tab to Calibri 12pt. This resets everything. Then reapply your heading styles.

File saved as Document1

Fix: File > Save As > choose OneDrive > Year 7 > DigiTech > Classwork > type the correct filename > Save. Check the title bar shows the right name afterward.

Table columns wrong size

Fix: Click inside the table, go to the Layout tab (appears when you're in a table) > AutoFit > AutoFit Contents. Or drag the column borders manually.

Red / blue squiggly lines

Red: Spelling error — right-click for suggestions. Blue: Grammar or style suggestion — right-click to see what Word recommends. Don't ignore them.
