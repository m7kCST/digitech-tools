💡 Starting a brand new file? Two ways to save it correctly
⭐ Best: Use Save As
  1. Open Word / PowerPoint / Excel
  2. Press Ctrl + Shift + S (or File → Save As)
  3. Navigate to the folder shown above
  4. Paste your filename in the name box
  5. Click Save — done!
🔁 Or: Rename after saving
  1. Save the file anywhere first
  2. Find it in File Explorer
  3. Click once → press F2
  4. Paste your filename → Enter
  5. Drag it into the correct folder
💡 Using OneDrive in a browser? Right-click the file → Rename → paste → Save.
