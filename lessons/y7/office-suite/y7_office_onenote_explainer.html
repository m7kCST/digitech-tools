What Even Is OneNote? 🤔

📓

A Digital Notebook

OneNote is like a physical ring binder - but digital. You can create as many notebooks, sections and pages as you need, and organise everything in one place.

☁️

Auto-Saves to OneDrive

OneNote saves automatically every few seconds to OneDrive. No Ctrl+S needed. Open it on your school laptop, home computer or phone and your notes are always there.

✏️

Type, Draw, Paste

Type notes anywhere on the page - not just in a fixed column. Paste images, draw diagrams, record audio, attach files. It is far more flexible than Word.

🔍

Searchable Everything

OneNote indexes all your notes so you can search across every notebook at once. Type a keyword and find the right page instantly - even handwritten notes.

Ctrl + F to search

How OneNote Is Organised 📚

Notebooks contain Sections, Sections contain Pages

📓
Notebook
The top level. One per year or purpose. e.g. "Year 7" or "Personal"
›
📑
Section
A coloured tab inside the notebook. One per subject. e.g. "DigiTech", "Maths"
›
📄
Page
A single note inside a section. One per lesson or topic. e.g. "Week 8 - Project Notes"
›
📦
Content Block
A box of content on a page. Text, image, table, drawing -- click anywhere to add one
💡 Suggested setup

Create one notebook called Year 7. Add a section for each subject. Inside each section, create a new page for each lesson or topic. Label pages with the week and topic -- e.g. Week 8 - 3D Printing Project.

Key Terms -- Know These 📍

Notebook
Year 7 / Personal

The outermost container. Think of it as a physical ring binder. You will usually only need one or two notebooks.

Section
DigiTech / Maths

A colour-coded tab inside a notebook. Create one section per subject. Colour-code them so you can find them instantly.

Page
Week 8 - Project Notes

A single canvas inside a section. Create a new page for each lesson or topic. Date your pages so you can find them later.

Content Block
Click anywhere to add

Click anywhere on the page and start typing -- that creates a content block. Drag it anywhere. This is different from Word where text flows in a fixed column.

Tags
To Do / Important / Question

Visual markers you can add to any line. The To Do tag adds a checkbox. You can search by tag to find all your to-dos across every notebook.

Quick Notes
Win + Alt + N

A mini OneNote window you can open instantly without opening the full app. Good for jotting something down quickly mid-lesson.

Ink / Draw
Draw tab

On a touchscreen or with a mouse, use the Draw tab to handwrite or sketch directly onto the page. Useful for diagrams, maths working and mind maps.

Sync
Auto -- no Ctrl+S needed

OneNote syncs to OneDrive automatically. You never need to save manually. If you see a sync error icon, click it and choose Sync Now.

What To Use It For 🎯

📝

Class Notes

Create a new page at the start of every lesson. Add the date, topic and any key points. Paste in screenshots from the board or class slides.

New page shortcut: Ctrl + N inside a section
🔬

Research

Copy and paste text and images from websites directly into OneNote. It automatically records the source URL so you can reference it later.

Right-click any image or text on a webpage and paste into OneNote
✅

To-Do Lists

Use the To Do tag (Ctrl + 1) to create checkboxes. Check them off as you complete tasks. Search by tag to see all your unfinished to-dos at once.

Home tab - Tags - To Do, or press Ctrl + 1
🗺️

Mind Maps and Diagrams

Click anywhere to place text boxes. Drag them around. Use the Draw tab to add arrows and connections. OneNote's free-form canvas is perfect for visual planning.

Draw tab - Pen or Shapes tools
📎

Attach Files

Insert tab - File Attachment. Attach a Word doc, PDF or spreadsheet directly into a OneNote page. Useful for keeping all resources for a topic in one place.

Insert tab - File Attachment
📸

Screenshots and Images

Paste any screenshot directly onto a page with Ctrl + V. Use Insert - Screen Clipping (Windows + Shift + S) to capture a part of your screen and insert it instantly.

Windows + Shift + S for a screen clipping

Tags -- Your Visual Markers 🏷️

✅
To Do Ctrl+1
Adds a checkbox. Check it off when done. Search all To Do tags to see unfinished tasks.
⭐
Important Ctrl+2
Marks something as key information. Good for exam tips, definitions, or things your teacher emphasised.
❓
Question Ctrl+3
Flag something you don't understand. Search for all Question tags before a test to review what needs clarifying.
💡
Remember for Later
Tag anything you want to revisit. Review all Remember tags at the end of a unit to build a revision summary.
📖
Definition
Mark key vocabulary. Search all Definition tags to build a glossary for any subject.
🔍
Search all tags
Home tab - Find Tags. Shows every tagged item across all your notebooks in one panel. Filter by tag type.

Do This / Never Do This 🚦

Never do this
  • Use OneNote for final submitted work
  • Leave pages unnamed or untitled
  • Dump everything onto one giant page
  • Ignore sync errors
  • Mix all subjects in one section
  • Use OneNote instead of Word for assignments
Always do this
  • One section per subject, colour coded
  • New page for each lesson or topic
  • Date every page at the top
  • Use tags to mark to-dos and questions
  • Use screen clippings for diagrams
  • Search notes before re-writing something

The Rules That Actually Matter ⭐

⭐ Golden Rules

1

OneNote auto-saves. You never need to press Ctrl+S. If you see a sync error, click the icon and select Sync Now -- do not ignore it.

2

One section per subject, colour coded. Setting this up once at the start of the year means you can find any note in seconds for the rest of the year.

3

Date every page. "Notes" is not a useful page title. "Week 8 - 3D Printing Project - 25 March 2026" is. Future you will thank present you.

4

OneNote is for notes -- Word is for final documents. Do not submit assignments written in OneNote. Transfer finished work to Word, format it properly, and save to OneDrive.

5

Use Ctrl+F before re-writing notes. Search your notebooks first -- you have probably already written it down somewhere.

When Things Go Wrong 🚨

🚨 Common Mistakes

Sync error icon

Fix: Click the sync error icon at the top of the page. Select Sync This Notebook. Check your internet connection. If it keeps happening, close and reopen OneNote.

Can't find a note

Fix: Press Ctrl+F and search by keyword. Make sure you're searching All Notebooks, not just the current section. If still not found, check the Recycle Bin (right-click the notebook name).

Accidentally deleted a page

Fix: Right-click the section or notebook name and select Notebook Recycle Bin. Pages are kept for 60 days before permanent deletion.

Content block in wrong place

Fix: Hover over the content block until the move cursor appears (four arrows). Click and drag it to the correct position on the page.

Notes not on other device

Fix: Make sure you are signed in with the same Microsoft account on both devices. Open OneNote and wait for it to sync -- it may take a minute if you have not opened it for a while.
