Week 1
What is a digital system?
Every computer works the same way. Data goes in, gets worked on, gets saved, and comes back out.
1 Input
Data comes IN
Keyboard, camera, microphone, touchscreen
tap to read more
"This is where I start! Input is anything that sends data into the computer. When you press a key - that's input. When you take a photo - that's input. When you tap the screen - that's input."
2 Process
The CPU does the thinking
The brain of the computer works on the data
tap to read more
"Inside the CPU is where the real work happens. The CPU runs billions of tiny calculations every second. Software tells the CPU what to do with me - sort me, change me, move me. It is incredibly fast."
3 Storage
Data gets saved
RAM for now, SSD or cloud for keeps
tap to read more
"There are two kinds of storage. RAM is fast but only works while the power is on - like your brain holding a thought. SSD or hard drive is permanent - like writing it in a book. Cloud storage is saved on a server far away."
4 Output
The result comes OUT
Screen, speakers, printer, or sent over network
tap to read more
"I made it out! Output is the result - what you see, hear, or get. A video playing on your screen is output. Music from speakers is output. A document sent to a printer is output. This is why the system exists!"

Week 2
Hardware and software
Hardware is the stuff you can touch. Software is the instructions. They need each other.
1 Hardware
Stuff you can touch
CPU, RAM, SSD, screen, keyboard, mouse
tap to read more
"Hardware is everything physical inside and outside the computer. The CPU is the brain. RAM is fast memory. The SSD saves things permanently. The screen, keyboard and mouse let you interact with it all."
2 Software
Instructions you can't touch
Windows, Word, Chrome, apps, games
tap to read more
"Software is the instructions that tell the hardware what to do with me. Without software, the hardware is just a box. The operating system (like Windows) runs everything. Apps like Word and Chrome are software too."
3 Performance
What makes it fast?
CPU speed, RAM, SSD, network - tap to explore
tap to read more
"A faster CPU = faster thinking. More RAM = more things open at once. An SSD loads files 5 times faster than an old hard drive. A fast network means files move quicker. All four affect how fast the system feels."

Week 3
Byte's big journey
Byte is now a Word document. Follow the journey from laptop to teacher - all the way through the cloud.
1 Input
Student types in Word
Keyboard sends data in. CPU runs Word software.
tap to read more
"I am born! Every key the student presses sends data into the system. The CPU runs Word's software and turns those keystrokes into a real formatted document. I live in RAM while the file is open."
2 Storage
File saved to the laptop
Ctrl+S writes Byte from RAM onto the SSD
tap to read more
"Safe! When the student presses Ctrl+S, I move from RAM onto the SSD. Now I will survive even if the power goes off. I am a real .docx file living on the device."
3 Transmission
Byte flies to the cloud!
Through wi-fi, through the internet, to OneDrive
tap to read more
"I am uploading! The file is broken into tiny packets of data and sent through the school's wi-fi, then through the internet, all the way to Microsoft's OneDrive servers. This is called transmission. It happens in seconds!"
4 Storage
Byte lives in the cloud
Saved on a server - accessible from anywhere
tap to read more
"I made it to the cloud! The cloud is not magic - it is just a huge building full of servers (computers) that store files for millions of people. I am saved here now. Anyone the student shares access with can reach me from any device."
5 Transmission
Teacher downloads the file
Byte travels back through the network
tap to read more
"The teacher opens OneDrive and clicks my file. I travel back through the internet and school network to the teacher's device. This is downloading - receiving a file from another system. Almost there!"
6 Output
Byte appears on screen!
The document displays on the teacher's monitor
tap to read more
"I MADE IT! The GPU renders me on the teacher's screen as a full document. This is output - the final result coming out of the system. The whole digital system worked: input, process, storage, transmission and output. Mission complete!"
